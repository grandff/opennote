# Extension Icons

이 폴더에 다음 크기의 아이콘 파일들을 추가해주세요:

- `icon16.png` - 16x16 픽셀
- `icon48.png` - 48x48 픽셀
- `icon128.png` - 128x128 픽셀

## 임시 아이콘 생성

개발 중에는 아래 명령어로 임시 아이콘을 생성할 수 있습니다 (ImageMagick 필요):

```bash
# 단색 아이콘 생성 (빨간색)
convert -size 16x16 xc:red icon16.png
convert -size 48x48 xc:red icon48.png
convert -size 128x128 xc:red icon128.png
```

또는 온라인 아이콘 생성기를 사용하세요:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
